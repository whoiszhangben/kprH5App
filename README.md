## 使用

```bash

# install & run
$ yarn & yarn dev
```

## 能力

- `eslint` 校检及错误提示
- 浏览器默认样式抹平
- 数据 `mock` 能力封装
- `prettier` 统一代码风格
- `vw` 移动端适配
- `vuex` 封装及使用
- `vue-router` 路由配置及权限控制
- `axios` 封装及请求
- 等等...

## 其他

1. 如果要将此项目用做 `PC` 端基础模板的话，则只需删除移动端适配相关代码，即 `postcss-px-to-viewport` 插件相关配置内容即可。

2. 地图json文件下载 http://datav.aliyun.com/portal/school/atlas/area_selector
